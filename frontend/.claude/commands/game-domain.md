# 건담 카드 게임 도메인 개념

출처: 건담 카드 게임 종합 규칙 Ver. 1.4.1 (2026년 1월 16일)

---

## 승패 조건

- **승리**: 상대 실드 에어리어가 0장인 상태에서 배틀 대미지 OR 상대 덱을 0장으로
- **패배**: 실드 0 상태에서 배틀 대미지 / 덱 0장(드로우 시점 포함) / 투료 선언

> 카드 텍스트 > 종합 규칙. 금지 효과 > 허용 효과.

---

## 덱 구성

| | 장수 | 구성 |
|---|---|---|
| 메인 덱 | 50장 | 유닛 / 파일럿 / 커맨드 / 베이스 (같은 카드 No. 최대 4장, 1~2색) |
| 리소스 덱 | 10장 | 리소스 카드만 (같은 카드 No. 제한 없음) |

- 카드 **플레이 조건**: 자신의 리소스 총 수 ≥ 카드 Lv. (액티브/레스트 불문)
- **코스트**: 액티브 리소스를 필요한 수만큼 레스트시켜 지불
- 리소스 에어리어 최대 15장 (EX 리소스 최대 5장 포함) / 패 상한 10장

---

## 카드 타입 (5종)

### 유닛
- 속성: AP(공격력), HP(내구력), 링크 조건
- 배치 직후 그 턴은 어택 불가 (링크 유닛 예외: 링크 조건 만족 파일럿 세트 시 즉시 어택 가능)
- HP 0 → 파괴 → 트래시
- 배틀 에어리어 상한 6기

### 파일럿
- 속성: AP 수정치(+N), HP 수정치(+N)
- 유닛 아래에 세트. 유닛 1기에 1명
- 카드 명 위 텍스트 = 파일럿 자신의 효과 (주로 【버스트】)
- 카드 명 아래 텍스트 = 세트된 유닛이 얻는 효과
- 파일럿의 특징은 유닛에 추가되지 않음. 임의 교체/제거 불가

### 커맨드
- 타이밍: 【메인】 또는 【액션】
- 플레이 후 트래시 (효과로 이동 지정 없을 경우)
- 【파일럿】 보유 시: 효과 발동 대신 파일럿으로 세트 가능
- 대상 선택 불가 시 플레이 불가

### 베이스
- 속성: AP, HP
- 실드 에어리어 베이스 존에 배치 (상한 1장)
- 베이스가 있는 동안 실드 에어리어 대미지는 베이스 우선 흡수
- HP 0 → 파괴 → 트래시

### 리소스
- 리소스 덱 → 리소스 에어리어에 배치. 코스트 지불에 사용
- Lv. 판정은 액티브/레스트 불문 총 수 기준

---

## EX 토큰

- **EX 베이스**: AP 0 / HP 3 베이스 토큰. 게임 시작 시 자동 배치
- **EX 리소스**: 코스트 지불 1회용. 후공 플레이어 1장 제공, 사용 후 제거

---

## 게임 영역 (존)

| 영역 | 번호 | 공개 | 설명 |
|------|------|------|------|
| 덱 에어리어 | ① | 비공개 | 메인 덱. 위에서 드로우 |
| 리소스 덱 에어리어 | ② | 비공개 | 리소스 덱 |
| 실드 에어리어 | ③⑥ | 혼합 | 베이스 존(공개 ⑥) + 실드 존(비공개 ③) |
| 리소스 에어리어 | ④ | 공개 | 최대 15장 |
| 배틀 에어리어 | ⑤ | 공개 | 유닛·파일럿 배치. 유닛 최대 6기 |
| 패(손패) | — | 비공개 | 상한 10장. 자신만 열람 |
| 트래시 | ⑦ | 공개 | 파괴·사용 카드. 순서 변경 가능 |
| 제외 에어리어 | — | 공개 | 제외된 카드. 파괴와 다른 처리 |

> 리소스·배틀·실드 에어리어를 합쳐 「필드」라고 부름.
> 카드가 영역 간 이동 시 특별한 지시 없이는 새로운 카드로 취급 (이전 효과 소멸).

---

## 게임 준비 순서

1. 덱 준비 (메인 50장 + 리소스 덱 10장)
2. 선공·후공 결정
3. 첫 패 드로우 (각 5장)
4. 멀리건 (선택 1회: 패 전체 반납 후 5장 드로우, 선공→후공 순)
5. 실드 배치 (각 6장, 뒷면)
6. EX 베이스 배치 (베이스 존에 토큰 1장)
7. EX 리소스 배치 (후공만)
8. 선공 플레이어의 스타트 페이즈부터 시작

---

## 턴 진행 흐름 (5 페이즈)

### 스타트 페이즈
1. **액티브 스텝**: 자신 필드(배틀·리소스·베이스 존)의 레스트 카드 전부 동시 액티브
2. **스타트 스텝**: 「턴 개시 시」 효과 발동

### 드로우 페이즈
- 덱 위에서 1장 드로우. 덱 0장 시 패배

### 리소스 페이즈
- 리소스 덱 위에서 1장을 리소스 에어리어에 앞면 액티브로 배치

### 메인 페이즈
아래 3가지를 원하는 순서로 반복. 「메인 페이즈 종료」 선언 시 엔드 페이즈로.
1. **패 플레이**: 코스트 지불 후 유닛 배치 / 베이스 배치 / 파일럿 세트 / 【메인】 커맨드 발동
2. **【기동·메인】 발동**: 유닛 어택 중이 아닐 때
3. **유닛 어택**: 액티브 유닛으로 상대 플레이어 또는 레스트 상대 유닛 어택

> 어택 중에는 패 플레이·기동메인 발동 불가.

### 엔드 페이즈
1. **액션 스텝**: 비 턴 플레이어부터 교대로 【액션】/【기동·액션】 발동. 양측 연속 패스로 종료
2. **엔드 스텝**: 「턴 종료 시」 효과 발동
3. **핸드 스텝**: 패 10장 초과 시 10장이 되도록 버림
4. **클린업 스텝**: 「이 턴 중」 효과 소멸. 해결 후 상대에게 턴 이동

---

## 배틀 (어택과 배틀 5스텝)

**어택 가능 조건**: 액티브 상태 자신 유닛. 배치 직후 그 턴은 불가 (링크 유닛 제외).
**대상**: 상대 플레이어 또는 레스트 상태의 상대 유닛.

1. **어택 스텝**: 액티브 유닛 1기를 레스트시키고 어택 대상 선언. 【어택 시】 효과 발동
2. **블록 스텝**: 비 턴 플레이어가 《블로커》 보유 유닛을 레스트시켜 어택 대상 변경 가능. 1회 어택에 1번
3. **액션 스텝**: 양측 교대로 【액션】/【기동·액션】 발동. 양측 연속 패스로 종료
4. **대미지 스텝**: 유닛끼리 어택 시 서로 AP만큼 배틀 대미지 교환. 플레이어 어택 시 실드에 순서대로 대미지
5. **배틀 종료 스텝**: 「이 배틀 중」 효과 전부 소멸. 메인 페이즈로 복귀

> 어택/블록 스텝 종료 시 유닛이 영역 이동했으면 다음 스텝 건너뛰고 배틀 종료 스텝으로.
> 《블로커》는 1회 어택에 1번. 원래 어택 대상 유닛은 《블로커》 발동 불가.

## 키워드 어빌리티

| 어빌리티 | 타이밍 | 설명 |
|----------|--------|------|
| 《리페어 N》 | 자신의 턴 종료 시 | N 회복. 여러 개 보유 시 수치 합산 |
| 《돌파 N》 | 자신 턴 중, 배틀 대미지로 상대 유닛 파괴 시 | 상대 실드 에어리어에 N 대미지 (베이스 우선). 실드/베이스 0장이면 발동 안 함 |
| 《원호 N》 | 【기동·메인】 이 유닛을 레스트 | 다른 아군 유닛 1기를 이 턴 중 AP +N |
| 《블로커》 | 블록 스텝 | 이 유닛을 레스트시켜 어택 대상을 이 유닛으로 변경. 1회 어택 1번, 한 유닛 다수 보유 불가 |
| 《선제공격》 | 대미지 스텝 | 상대보다 먼저 배틀 대미지 부여. 상대 유닛/베이스 파괴 시 반격 대미지 없음. 한 유닛 다수 보유 불가 |
| 《고기동》 | 어택하는 동안 상시 | 이 유닛이 어택하는 동안 상대 유닛은 《블로커》 발동 불가. 한 유닛 다수 보유 불가 |
| 《제압》 | 플레이어 어택 시 | 실드에 대미지 시 위에서 2개 실드에 동시 대미지. 실드 1개뿐이면 1개만. 한 유닛 다수 보유 불가 |

## 키워드 트리거

| 트리거 | 설명 |
|--------|------|
| 【기동·메인】 | 메인 페이즈(어택 중 제외)에 조건 만족 시 발동하는 기동 효과 |
| 【기동·액션】 | 액션 스텝에 조건 만족 시 발동하는 기동 효과 |
| 【메인】 | 커맨드 카드 전용. 메인 페이즈(어택 중 제외)에 플레이 |
| 【액션】 | 커맨드 카드 전용. 액션 스텝에 플레이 |
| 【버스트】 | 실드 파괴 시 앞면이 되었을 때 코스트 없이 발동 가능. 선택. 트래시 놓이기 전 발동. 다른 유발 효과보다 최우선 |
| 【배치 시】 | 카드가 배치되었을 때 |
| 【어택 시】 | 유닛이 어택 스텝에 어택 선언 시 |
| 【파괴 시】 | 유닛/베이스가 파괴되어 트래시에 놓였을 때. 트래시에서 발동. 파괴 직전 필드 상태 참조 |
| 【세트 시】 | 유닛에 파일럿이 세트되었을 때 |
| 【세트 중】 | 파일럿이 세트되어 있는 동안 |
| 【링크 시】 | 링크 조건 만족 파일럿이 세트되었을 때 |
| 【링크 중】 | 링크 조건 만족 파일럿이 세트되어 있는 동안 |
| 턴 1회 | 그 턴 중 1번만 발동. 동일 효과 카드가 여러 장이어도 각각 1번씩 |

## 효과의 종류

| 종류 | 설명 |
|------|------|
| 상시 효과 | 항상 발동 중. 즉시 적용. 모순 시 금지 효과 우선 |
| 유발 효과 | 특정 사건 발생 시 자동 유발. 동시 유발 시 턴 플레이어 먼저. 새 유발은 새것 먼저. 【버스트】 최우선 |
| 기동 효과 | 플레이어가 임의로 발동. 「조건 : 효과」 형식 |
| 커맨드 효과 | 커맨드 카드를 지정 타이밍에 플레이해 발동. 대상 선택 불가 시 플레이 불가 |
| 치환 효과 | 「(A)하는 대신 (B)한다」 형식. 사건 A를 B로 대체 |

> 「한다」→ 가능한 한 처리. 「해도 좋다」→ 발동 여부 선택.
> 「그렇게 했다면」→ 앞 문장 미해결 시 뒷 문장 해결 불가. 「그 후」→ 앞 문장 결과 무관 뒷 문장 해결 가능.

## 주요 용어

| 용어 | 정의 |
|------|------|
| 액티브 / 레스트 | 세로로 놓인 상태 / 가로로 놓인 상태. 필드·리소스·베이스 존 카드에 적용 |
| 배치 | 유닛이나 베이스를 필드에 놓는 것 |
| 세트 | 파일럿 카드(또는 【파일럿】 커맨드)가 유닛 아래에 겹쳐지는 것 |
| 파괴 | HP ≥ 대미지 or 효과로 필드에서 트래시로. 실드 파괴 시 앞면으로 해 【버스트】 확인 후 트래시 |
| 제외 | 어느 영역에서 제외 에어리어로 이동. 파괴와 다름 |
| 버리다 | 패에서 트래시로 이동 |
| 드로우 | 덱 맨 위 카드를 비공개로 패에 추가 |
| 플레이 | 패의 카드를 공개하고 코스트를 지불해 사용 |
| 회복 | 유닛/베이스의 대미지 카운터를 지정 수만큼 제거. HP 이상으로 회복 불가 |
| 배틀 대미지 / 효과 대미지 | 배틀 결과 대미지 / 카드 효과 대미지 |
| 턴 플레이어 | 현재 진행 중인 턴의 플레이어 |
| A/B (슬래시) | A와 B 어느 쪽이든 해당 |
