# 배틀 상세 규칙

## 어택 조건

- **액티브 상태**의 유닛만 어택 가능
- 어택 선언과 동시에 해당 유닛을 **레스트**시킴
- 배치된 그 턴은 어택 불가 (링크 유닛 예외)
- 대상: 상대 플레이어 **또는** 레스트 상태의 상대 유닛

## 레스트 복구 타이밍

- 자신의 **스타트 페이즈 액티브 스텝**에 자신 필드 전체 액티브 (배틀·리소스·베이스 존)
- 레스트된 유닛은 상대방 턴에는 액티브로 돌아오지 않음
- 따라서 어택한 유닛은 **다음 자신의 턴 시작까지** 레스트 유지

## 상대 유닛 어택 타이밍

- 상대 턴의 메인 페이즈에 상대가 자신의 액티브 유닛으로 어택
- 상대 유닛도 어택 선언 시 레스트됨 → 자신의 다음 턴 시작까지 레스트
- 상대도 동일하게 **액티브 유닛이 남아있는 한 반복 어택 가능**

## 직접 어택 시 대미지 처리 순서

상대 플레이어에게 어택 시:

1. **베이스가 있으면** → 베이스에 AP만큼 대미지 (베이스 HP 0 → 파괴 → 트래시)
2. **베이스 없고 실드 있으면** → 실드 1장을 앞면으로 뒤집어 트래시 (【버스트】 확인)
3. **실드 0 + 베이스 없음** → 직격 → 즉시 패배

## 유닛 vs 유닛 대미지 처리

블로커가 있거나 레스트 유닛을 대상으로 어택 시:

- 양측이 **동시에** 서로 AP만큼 배틀 대미지 교환
- HP 0 이하가 된 유닛은 **파괴** → 트래시
- 《선제공격》 보유 시: 내가 먼저 대미지 적용 → 상대 유닛 파괴되면 반격 대미지 없음 / 생존 시 반격

## 블록 스텝 규칙

- 비 턴 플레이어(수비 측)가 《블로커》 보유 유닛을 **레스트**시켜 어택 대상을 해당 유닛으로 변경
- 1회 어택에 1번만 발동 가능
- **원래 어택 대상 유닛**은 《블로커》 발동 불가
- 《고기동》 보유 유닛이 어택하는 동안 상대는 《블로커》 발동 불가

## 《돌파 N》 발동 조건

- 자신 턴 중, **배틀 대미지로 상대 유닛을 파괴**했을 때만 발동
- 상대 실드 에어리어에 N 대미지 (베이스 우선)
- 상대 실드도 베이스도 0장이면 발동 자체가 일어나지 않음

## 《버스트》 발동 타이밍

- 실드가 파괴되어 **앞면이 되었을 때** 코스트 없이 발동 가능 (선택)
- 트래시에 놓이기 전 발동
- 다른 유발 효과보다 최우선

## 승패 조건

| 조건                                             | 결과      |
| ------------------------------------------------ | --------- |
| 상대 실드 0 + 베이스 없음 상태에서 배틀 대미지   | 즉시 승리 |
| 자신의 실드 0 + 베이스 없음 상태에서 배틀 대미지 | 즉시 패배 |
| 덱 0장에서 드로우                                | 패배      |
| 투료 선언                                        | 패배      |

## 배틀 종료 스텝

- 배틀 중 발동한 효과는 이 스텝에서 전부 소멸
- 어택/블록 스텝 도중 유닛이 영역 이동하면 이후 스텝 건너뛰고 배틀 종료 스텝으로 직행
